# 😊 Happiness Survey Analysis

**Course:** MBA Business Data Analytics  
**Tools:** R · ggplot2 · dplyr · regression modeling · classification analysis  
**Dataset:** Survey-based dataset with demographic, behavioral, and satisfaction variables

---

## Business Problem

What factors are most strongly associated with happiness and life satisfaction?  
This project uses regression and classification analysis to examine how survey responses, neighborhood satisfaction, and other personal factors influence reported happiness outcomes.

---

## What I Did

**1. Data Preparation**  
- Loaded and explored the survey dataset in R
- Reviewed variable types, distributions, and missing values
- Prepared variables related to life satisfaction, neighborhood perceptions, and personal characteristics
- Structured data for both regression and classification analysis

**2. Regression and Classification Modeling**  
- Built multiple linear regression models to explain happiness outcomes
- Applied logistic regression to evaluate binary or categorized satisfaction outcomes
- Assessed coefficient significance, model fit, and classification performance
- Compared how different variables contributed to perceived happiness

**3. Interpretation and Insight**  
- Examined how neighborhood satisfaction influenced broader happiness measures
- Identified which variables showed the strongest relationships with self-reported well-being
- Translated model output into practical behavioral and social insight

---

## Key Insight

Neighborhood satisfaction emerged as an important predictor of happiness, showing that environmental and perception-based factors can meaningfully influence self-reported well-being.

---

## Business Value

- Demonstrated how analytics can be applied to behavioral and survey-based data
- Showed how regression and classification methods complement one another
- Identified important drivers of happiness and satisfaction outcomes
- Reinforced the use of analytics for social and behavioral insight generation
